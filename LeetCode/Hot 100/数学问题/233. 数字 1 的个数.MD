    //枚举每一数位上 1 的个数
    //以n=1234567 为例，我们需要统计「百位」上数字 1 出现的次数。
    //对于从 0 开始每 1000 个数，「百位」上的数字 1 都会出现 100 次，即数的最后三位每 1000 个数都循环[100, 199] 100 次。n 拥有1234个这样的循环，一共出现1234×100 次 1。
    //使用公式表示，完整循环百位出现的 1 的次数为：n/1000*100;即n/(10*div)*div
    
    //剩余部分567中百位为1的次数：
    //当n′<100 时，「百位」上不会出现 1；当 100≤n′<200 时，「百位」上 1 出现了 n'-100+1 次 1；n′≥200 时，「百位」上出现了全部100 次 11。  
    //min(max(n′−100+1,0),100)；    
    
    public int countDigitOne(int n) {
        long div = 1;
        int ans = 0;
        while(div<=n){
            ans+=n/(10*div)*div+Math.min(Math.max(n%(10*div)-div+1,0),div);
            div*=10;
        }
        return ans;
    }
